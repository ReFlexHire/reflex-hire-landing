2. 관계형 데이터베이스 외래키 설정
3. 인덱스 최적화
4. 마이그레이션 스크립트 작성

```sql
CREATE TABLE jobs (
  id SERIAL PRIMARY KEY,
  client_id INTEGER REFERENCES users(id),
  title VARCHAR(255) NOT NULL,
  salary_range INTEGER,
  status VARCHAR(50) DEFAULT 'active',
  created_at TIMESTAMP DEFAULT NOW()
);
```
