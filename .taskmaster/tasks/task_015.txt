2. 리크루터용 후보자 제출 인터페이스

3. 채용 진행 상태 관리 (지원→면접→최종합격)
4. 알림 시스템 (이메일/인앱 알림)

```javascript
// 채용 요청 등록 API
app.post('/api/jobs', checkRole(['CLIENT']), async (req, res) => {
  const { title, salary_range, requirements } = req.body;
  const job = await db.query(
    'INSERT INTO jobs (client_id, title, salary_range, requirements) VALUES ($1, $2, $3, $4) RETURNING *',
    [req.user.id, title, salary_range, requirements]
  );
  res.json(job.rows[0]);
});
```
